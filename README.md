# node-cli
cli app to convert log dile to .txt and .json file

## Setup
install dependencies
```
npm i -g
```
use flag -g to register app globally

## Usage Example
```
fazacli /var/log/syslog
```

## Help
use this for complete usage instructions
```
fazacli -h
```
or
```
fazacli --help
```
